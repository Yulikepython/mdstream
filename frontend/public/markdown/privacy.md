# 個人情報保護方針

**MDStream**（以下、「本サービス」といいます）における個人情報の取り扱いについて、以下のとおり方針を定めます。

## 1. 個人情報の定義
「個人情報」とは、氏名、メールアドレスなど、生存する個人を特定できる情報をいいます。また、他の情報と容易に照合でき、それにより個人を識別できるものも含まれます。

## 2. 利用目的
本サービスでは、ユーザーの個人情報を以下の目的で利用します。
- 本サービスへのログイン・認証
- ユーザーサポートや通知
- ドキュメントの作成・閲覧権限の管理

## 3. 第三者への提供
法令に定めがある場合を除き、ユーザーの個人情報を第三者に開示または提供しません。

## 4. セキュリティ対策
本サービスは、個人情報への不正アクセス・紛失・改ざん等を防止するため、セキュリティ対策を講じます。

## 5. クッキー（Cookie）の利用
ユーザーが本サービスを利用する際に、Cookieや類似の技術を利用する場合があります。Cookie情報はブラウザ設定で拒否できますが、一部機能が制限される場合があります。

## 6. 改訂
本方針は、法令や本サービスの運用状況に応じて適宜見直し・改訂を行う場合があります。変更後の方針は、本サービス上に掲載した時点から効力を有します。
